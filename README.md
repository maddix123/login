# CV2 Shell App (VPS Install)

MODS99 Passport Photo Tool packaged as a shell app for VPS deployment.

## One-line install

```bash
curl -fsSL https://raw.githubusercontent.com/maddix123/cv2/main/install.sh | sudo bash
```

## Run

```bash
cv2
```

By default it runs on port `8080`.

Open in browser using your VPS IP and port:

```text
http://YOUR_SERVER_IP:8080
```

You can change the port:

```bash
PORT=9090 cv2
# then open http://YOUR_SERVER_IP:9090
```
