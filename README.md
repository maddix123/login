# CV2 Shell App (VPS Install)

This repo packages your passport-guide page as a shell app you can install on a VPS.

## One-line install

```bash
curl -fsSL https://raw.githubusercontent.com/maddix123/cv2/main/install.sh | sudo bash
```

## Run

```bash
cv2
```

Default port is `8080`. To change it:

```bash
PORT=9090 cv2
```
